## Task Management Tool (In Progress)
For assigning, tracking, and reviewing employee tasks.

### Key Features
*   Create and assign tasks to team members.
*   Track task progress (To Do → In Progress → Done).
*   Set due dates and reminders.
*   Allow comments, attachments, and priority levels.
*   Dashboard view for managers (team performance overview).
*   Integration with Performance Management System (below).

## Performance Management System (Integrated with Task Tool)
Tracks performance metrics, incentives, and appraisals.

### Key Features
*   Auto-calculate monthly performance scores based on task completion, quality rating, or attendance.
*   Auto-generate incentive & appraisal reports (monthly/quarterly).
*   Employee dashboard showing their performance summary.
*   Manager dashboard for evaluation and notes.
*   Export reports for HR/payroll.

## Reminders & Notifications Module
Automated reminders for important people and company events.

### Key Features
*   **Employee Birthday & Work Anniversary Alerts**
    *   Automatic reminders to HR and team leads.
    *   Option to auto-send wishes (email or Slack/WhatsApp message).
*   **Holiday Calendar**
    *   Shows upcoming holidays.
    *   Sends reminders a day before.
*   **Important Event Reminders**
    *   HR or Admin can manually create reminders (e.g., audits, meetings, training sessions).
